########################## Simple Rain Alert App with SMS - Twilio #####################

Modules Used:<br/>
    - os<br/>
    - twilio<br/>
    - requests<br/>

Description:<br/>
    - Allows you to send sms to a specific person about the rain forecast for a specific location set<br/>
    - Use of Open Weather API<br/>
    - Use of Twilio Messaging service<br/>
